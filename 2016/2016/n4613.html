Networking TS - Editors Report

History

N4612 is the third working draft for the Networking TS, revising N4588. All changes are editorial, fixing spelling, grammar and formatting. Some design changes were applied to the LaTeX sources, but were reverted for N4612 and proposed to LWG in P0405R0 instead.

Complete indices have been added for all terms and library names. Any missing entries are accidental and should be fixed.

Significant editorial changes:

  • Use fixed-width font for POSIX identifiers.
  • Spelling fixes ("sub-clause", "life cycle", "asynchronous", "canceling", "canceled", "successfully").
  • Further typographical fixes in Table 10 (async_result semantics) and Table 27 (GettableSocketOption requirements)
  • Rename placeholder variables in Tables 29 and 30 for consistency.
  • [socket.streambuf] Remove redundant template argument list in basic_socket_streambuf signatures, using injected-class-name instead.
  • [socket.algo.async.connect] Add missing paragraph break, and formatting.
  • [internet.address.v4.const].Add missing braces.
  • [internet.resolver.ops] Remove stray reference from get_executor return type.
  • [socket.acceptor.ops]: fix default arg to listen; max_connections -> max_listen_connections
  • [fwd.decl.synop] Fix declarations of address iterator and range types
  • [buffer.dynamic.string] Wrong arguments passed to basic_string::erase().
  • [socket.streambuf.cons] Use time_point::max, not clock_type::max.
